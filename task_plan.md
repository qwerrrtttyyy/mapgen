## 决策记录
- 使用 Web Worker 运行地图生成，保持主线程响应。
- 主题系统使用 CSS 自定义属性 `--mg-*`，支持 Aurora / Parchment。
- 安全策略采用 CSP meta 标签 + DOM 构建替代 innerHTML。
- 数据结构优先实现，供后续算法与缓存使用。
